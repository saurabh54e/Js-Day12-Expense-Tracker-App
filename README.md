# 🚀 Day 12/30 – Expense Tracker Web App

Continuing my 30 Days JavaScript Challenge, I built an Expense Tracker Web App using Vanilla JavaScript and Local Storage.

## Tech Used

* HTML
* CSS
* Vanilla JavaScript
* Local Storage API

## Features

* Add income and expense transactions
* Edit and delete existing transactions
* Automatically calculate current balance
* Track total income and total expenses
* Categorize transactions for better organization
* Store transaction history using Local Storage
* Persistent data even after page refresh
* Responsive and user-friendly UI

## What I Learned

* Working with DOM manipulation and event handling
* Implementing CRUD operations (Create, Read, Update, Delete)
* Using JavaScript array methods like `filter()`, `reduce()`, `forEach()`, and `findIndex()`
* Managing application state using objects and arrays
* Working with the Local Storage API for data persistence
* Building dynamic and interactive JavaScript applications
* Updating the UI efficiently based on user interactions

## Next Improvements

* Search transactions by title
* Filter transactions by category
* Sort transactions by amount or date
* Add monthly expense analytics and charts
* Export transactions as CSV files
* Dark mode support
* Budget tracking and spending insights
